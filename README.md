# Лабораторная работа: Основы PHP (Циклы, Массивы, Файловая система)

## Инструкции по запуску проекта

1. Клонируйте репозиторий и перейдите в папку проекта:\
   ```
   git clone https://github.com/pnchsps/php-lab-4 \
   cd php-lab-4
   ```
3. Убедитесь, что установлен PHP 7.4 или выше.

4. Запустите встроенный веб-сервер:
```
php -S localhost:8000
```
5. Откройте в браузере:

  Главная страница: http://localhost:8000/index.php

  Задание 4: http://localhost:8000/task4.php

  Задание 5: http://localhost:8000/task5.php

Примечание: Для корректной работы галереи в task5.php рядом с файлом должна лежать папка image/ с 20–30 .jpg изображениями.

## Описание лабораторной работы

В ходе этой работы были отработаны ключевые приёмы работы с PHP:

1. Циклы: использование for и while, вывод промежуточных и итоговых значений.

2. Массивы: генерация простых и ассоциативных массивов, вывод через print_r.

3. Функции: написание пользовательских функций, применение array_map() и array_reduce().

4. Классы: определение класса Transaction, методы для расчёта общей и средней суммы.

5. Файловая система: сканирование папки через scandir(), фильтрация по расширению, динамическая генерация HTML-галереи.

### Краткая документация к проекту
1. Задание 1: Цикл for (итерации 0–5, +10 к $a, +5 к $b).

2. Задание 2: Цикл while с теми же операциями.

3. Задание 3: Генерация массива из 10 случайных чисел от 1 до 100 и вывод через
```
<pre>print_r()</pre>
```
4. Задание 4: 

Ассоциативный массив транзакций (5 записей).

Функции:

calculateTotalAmount(array): float — сумма всех транзакций.

calculateAverage(array): float — средняя сумма.

mapTransactionDescriptions(array): array — массив описаний.

Класс Transaction с методами calculateTotal() и calculateAverage() для массива объектов.

5. Задание 5: 

Сканирование директории image/ (scandir()).

Фильтрация только JPG/JPEG-файлов.

Вывод галереи через CSS Grid (3 колонки).

Хедер с меню, основной блок и футер с динамическим годом (date('Y')).

### Примеры использования:
Запуск и открытие главной страницы
```
php -S localhost:8000
```
В браузере открыть:
```
http://localhost:8000/index.php
```
Результат цикла for (Задание 1)
```
for ($i = 0; $i <= 5; $i++) { 
    $a += 10; 
    $b += 5; 
    echo "Итерация {$i}: a = {$a}, b = {$b}<br>"; 
}
```
Галерея изображений (Задание 5)
```
$files = scandir('image/'); 
$images = array_filter( 
    $files, 
    fn($f) => in_array( 
        strtolower(pathinfo($f, PATHINFO_EXTENSION)), 
        ['jpg','jpeg'] 
    ) 
); 
foreach ($images as $img) { 
    echo "<img src='image/{$img}' alt='{$img}'>"; 
} 
```
